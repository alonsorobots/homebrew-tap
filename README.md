# Homebrew Tap for alonsorobots

Custom Homebrew formulae and casks.

## Install

```bash
brew tap alonsorobots/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `videobricks` | Convert videos to high-quality GIFs and MP4s |

### VideoBricks

```bash
brew install --cask alonsorobots/tap/videobricks
```

Or after tapping:

```bash
brew install --cask videobricks
```

For more info, see the [VideoBricks repo](https://github.com/alonsorobots/VideoBricks).
